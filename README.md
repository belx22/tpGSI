# Application de gestion d'une boulangerie

## Description
Cette application web permet de gérer les activités quotidiennes
d’une boulangerie telles que la gestion des produits, des ventes,
des stocks et du personnel.

## Fonctionnalités
- Gestion des produits (pain, viennoiseries, pâtisseries)
- Gestion des stocks
- Enregistrement des ventes
- Gestion des employés
- Suivi des recettes journalières

## Technologies utilisées
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript
- XAMPP

## Installation
1. Installer XAMPP
2. Démarrer Apache et MySQL
3. Importer la base de données
4. Copier le projet dans le dossier `htdocs`
5. Accéder à l’application via le navigateur
